Modern and Simple Static Site Generator written in PHP

Statie takes HTML, Markdown and Twig or Latte files and generates static HTML page.

See Docs GitHub

or see what have our community prepared. 

$ composer require symplify/statie
$ vendor/bin/statie generate source

  Website was successfully generated to /output.

$ php -S localhost:8000 -t output

  Open browser at http://localhost:8000 to see it.

Features

Markdown and Twig or Latte

Create your content easily in Markdown and your templates with the use of Twig or Latte.

Symfony-like configuration

Use yaml configuration files to set globally accessible parameters and service autowiring.

PHP 7.1

Built with the newest version under the hood so you can be sure you work with the most reliable ally.
